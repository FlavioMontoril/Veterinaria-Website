# Landing Page Veterinária

Landing page desenvolvida para uma médica veterinária especializada em suinocultura.

## 🚀 Tecnologias utilizadas

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome
- Lucide Icons

## ✨ Funcionalidades

- Layout responsivo
- Menu mobile
- Carousel de imagens
- Navegação com scroll suave
- Seções institucionais
- Formulário de contato
- Footer com redes sociais

## 📱 Responsividade

O projeto foi desenvolvido com foco em experiência responsiva para dispositivos móveis e desktop.
